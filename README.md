**Download zip**

```bash

```

## Buka di kode editor


## Install composer

```bash
composer install
```

## Copy .Env

```bash
copy .env.example.env
```

## Buka Web Server


## Buat database di localhost 

```bash
nama database :buku
```

## Setting database di .env

```bash
DB_PORT=3306
DB_DATABASE=buku
DB_USERNAME=root
DB_PASSWORD=
```

## Generate key

```bash
php artisan key:generate
```

## Jalankan migrate dan seeder

```bash
php artisan migrate --seed
```

## Buat Storage Link

```bash
php artisan storage:link
```
